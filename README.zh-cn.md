# 上古卷轴5护甲值计算器 (Skyrim Armor Rating Calculator)

[Read in English](README.md) | [Leia em Português](README.pt-br.md)

**目标：** 模拟《上古卷轴5：天际》的护甲值系统，根据装备的护甲部件计算减伤。

在《天际》中，许多玩家并不了解护甲系统背后的实际运作机制。本项目用 C 语言重构了该逻辑，考虑了显示护甲值、隐藏护甲奖励以及 80% 的减伤上限。

## 原理

减伤计算公式为：

```
减伤百分比 % = (显示护甲值 + 隐藏护甲值) × 0.12
```

### 核心概念

*   **隐藏护甲值：** 每件装备的护甲（头盔、胸甲、护手、靴子）都会提供 **25 点** 的隐藏奖励，全套装备总计 **100 点隐藏护甲值**。
*   **护甲上限 (Armor Cap)：** 《天际》的最大物理减伤上限为 **80%**。要达到此上限，你需要总计 **667** 点护甲值。如果装备了 4 件护甲，由于有隐藏奖励，你只需要 **567 点显示护甲值** 即可达到上限。

## 如何运行

你可以使用 `CMake` 编译本项目：

```bash
mkdir build
cd build
cmake ..
make
./bin/Armor <伤害值>
```

**示例：**

```bash
./bin/Armor 100
```

程序会要求你输入 4 个装备部件的护甲值，然后计算你的总护甲值、减伤百分比以及最终受到的伤害。

## 函数说明

*   **argc_verify:**
    *   **参数：** `int max_argc`, `int argc`, `char **argv`
    *   **返回：** `void`
    *   **过程：** 验证提供的命令行参数数量是否正确。如果不正确，打印正确的使用格式并退出程序。

*   **equip_armor:**
    *   **参数：** `armor *armor_equipped`
    *   **返回：** `void`
    *   **过程：** 遍历 4 个护甲槽位，提示用户输入每件护甲的护甲值。使用 `do-while` 循环确保输入有效（非负整数）。

*   **calculate_total_char_armor_rating:**
    *   **参数：** `armor *armor_equipped`
    *   **返回：** `int` (显示的总护甲值)
    *   **过程：** 汇总所有装备部件的护甲值并返回显示的总护甲值。

*   **calculating_damage_reduction:**
    *   **参数：** `int total_armor_rating`
    *   **返回：** `float` (十进制减伤值)
    *   **过程：** 应用公式 `(total_armor_rating × 0.12) / 100` 计算减伤百分比。如果结果超过 80% 的上限，则取值为 0.80。
